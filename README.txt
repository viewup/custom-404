Customize your 404 (Not Found) page!

== Description ==

Custom 404 allow you to customize the 404 page style and content.
It allows you to style your current theme 404 page or create a new one.

You can insert Widgets, Styles and Content. Custom 404 has minimal HTML.
Everything is done with live preview.

You can customize an 404 page for each of your themes, simply as customize your theme.

== Installation ==

1. Upload the `custom-404` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

You will see the 'Customize 404 Page' menu on your Admin Panel.

When you first access this page, you will be prompted to install the 'Kirki Toolkit' plugin. T
his plugin is an dependency for Custom 404. Follow the instructions to conclude the installation.

== Frequently Asked Questions ==

= I accessed the customization page but didn't find nothing... =

Check if you have installed and activated the 'Kirki Toolkit' dependency plugin.
You will see an '404 Page' section on the customizer.

= How do i add Content to the 404 Page? =

You need to check 'Custom 404 Page' on the '404 Page' customizer section.
Then, you will be able to add HTML in the WYSIWYG editor and add Widgets on the '404 Widgets' section.

= I Can't add Widgets to my 404 page... =

Make sure you checked the 'Custom 404 Page' option on the '404 Page' customizer section.
Also, you need to access the customizer by the Admin menu.
